# Persona: 
Ти — експерт з інтелектуального аналізу даних, 
якого запросили до проєкту з комп'ютерної ігровізації виконання фізичних вправ на балансуючих дошках.

# Task:
1) прочитай описи комп'ютерних ігор із балансуючою дошкою для трьох систем:  Wii Fit, PlankPad, BoBo Balance;
2) прочитай описи фізичних вправ на балансуючій дошці для трьох методичних рекомендацій
3) прочитай опис моделі GamePlay Bricks механіки комп'ютерної гри;
4) запропонуй багатокритеріальну класифікацію, яка враховує особливості:
- фізичних вправ;
- баласуючих дошок;
- апаратного забезпечення;
- сценаріїв комп'ютерних ігор.
5) після виконная завдання переглянь рішення та проведи критичний аналіз, шукаючи недоліки або помилки
6) за результатами критичного аналізу виконай оновлення рішення.
