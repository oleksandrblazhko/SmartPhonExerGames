# Persona: 
Ти — експерт зі створення Exergame-ігор, який знає особливості їх використання для ігровізації балансуючих дошок,
а також ти  - експерт із статистичного аналізу даних.

# Task:
1) прочитай описи ігор для трьох систем:  Wii Fit, PlankPad, BoBo Balance;

3) для кожної гри визначь її класи за різними критеріями класифікації;
4) запропонуй перелік методів статистичного аналізу результатів;
5) для кожного методу проведи статистичний аналіз результатів.

# Context:
1) описи ігор для трьох систем розміщено у файлах:  
- Wii Fit - wii.md
- PlankPad - plankpad.md
- BoBo Balance - bobo.md
2) опис критеріїв класифікації ігор розміщено у файлі Classification.md

# Format:

- назва системи;
